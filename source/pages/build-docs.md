# Recreating the Documentation

Follow these step-by-step instructions to rebuild, preview, and generate this integrated documentation site locally on your computer.

---

### Environment Setup

Before running the compilation tools, you must activate the dedicated Python environment containing Sphinx and the Markdown parsing extensions.

1. Open your terminal or Anaconda Prompt.
2. Run the following command to activate the environment sandbox:
   ```bash
   conda activate doc_hub_env
