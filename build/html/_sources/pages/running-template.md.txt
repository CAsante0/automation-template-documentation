# Terminology & Glossary

Before diving in, here is a quick translation guide for the common terms used throughout this project and its logs.

### Key Concepts

* **Environment (or Virtual Environment):** > An isolated digital "sandbox" on your computer. It ensures this project uses the exact software versions it needs without messing up your other applications.
* **Dependency:** > A pre-built software tool or package (written by someone else) that our project relies on to perform tasks like math, graphing, or data loading.
