src.app.util.missing_files.find_missing_dates_inferred(directory_path)

Scans a directory for GLORYS files, extracts hyphenated dates (YYYY-MM-DD) from filenames, infers the start/end dates from the file set, and identifies which dates within that range are not represented by a file.

The file format is assumed to be: GLORYS_REANALYSIS_…_{DATE_STRING}_alldays.nc

Parameters:

directory_path – The path to the directory to scan.

Returns:

A list of dates (as strings) that are missing a corresponding file.
